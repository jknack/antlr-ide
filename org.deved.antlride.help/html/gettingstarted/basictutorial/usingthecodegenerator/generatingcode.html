Generating code

In this section, you will learn how to generate code from your grammar

The code is generated every time that you save your grammar. The code generation is integrated in the build phase. If the build phase pass you'll see the generated resources in the output location, otherwise the generated resources (if any) will be deleted

  1. Double click in Expr.g file to open it with the ANTLR editor
  2. Make any change and then save it
