Using rule templates

In this section you will use content assist to create a Java like identifier matching rule.

  1. Create a new grammar. For example: G.g. See creating your first grammar.

  2. Delete the rule "emptyRule"
  3. Press Ctrl+Space to activate the content assist. The content assist window with a list of proposals will appear. Scroll the list to see the available choices.
  4. Type "j" and select the template "javaId"

  5. Press Ctrl+Space, type "j" and select the "javaIdPart".
  6. Press Ctrl+Space, type "j" and select the "javaIdStart".

  7. Save the file.
