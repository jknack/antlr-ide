Migrating from 1.3.1

  • Close all the grammar files opened
  • Close the ANTLR Interpreter View
  • Go to Run > Run Configurations... select the "ANTLR Java Application" option. Delete all the launches configurations
  • Close the ANTLR Interpreter View
  • You can't use the test cases created with ANTLR 1.3.1. You will need to create them again
  • The "Generation code" action is no longer available. The code generation is triggered when you save your grammar
